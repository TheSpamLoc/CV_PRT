Despliegue de Servidor NAS para Backups Centralizados

NAS OpenMediaVault Linux Rsync Automatización

Diseñé e implementé una solución de almacenamiento en red (NAS) utilizando OpenMediaVault para centralizar y proteger los datos críticos. Configuré un sistema de respaldos diarios automatizados para todas las estaciones de trabajo mediante rsync, garantizando la integridad y la rápida recuperación de la información ante cualquier incidencia.

Solución Wi-Fi Corporativa con Autenticación Centralizada

Wireless RADIUS Windows Server Cisco WLC Seguridad

Implementé una solución de red inalámbrica unificada para toda la empresa, cubriendo oficinas, bodegas y salas de venta. El proyecto se basó en un Cisco Wireless LAN Controller (WLC) integrado con un servidor RADIUS en Windows Server, lo que permitió centralizar la autenticación de usuarios y aplicar políticas de seguridad consistentes en todas las áreas del negocio.

Implementación de Perímetro de Seguridad con pfSense

pfSense Firewall XenServer VPN VLANs SquidGuard

Diseñé y desplegué una solución de seguridad perimetral robusta, instalando pfSense como firewall principal sobre un entorno virtualizado con XenServer. Se implementaron VPNs para el acceso remoto seguro y se segmentó la red interna mediante VLANs con listas de acceso (ACLs) para un control granular del tráfico. Además, se configuró el filtrado de contenido web con SquidGuard para proteger la red de amenazas y aplicar políticas de uso de internet.
